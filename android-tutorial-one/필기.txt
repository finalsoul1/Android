Android 4대 구성요소
1. Activity
 - ~.jave : 상태정보(변수), 이벤트 함수(ex 버튼 클릭)
 - ~.xml : HTML 처럼 태그로 화면을 구성

2. Service
 - 화면은 없지만 로직을 별도로 제공해야 함

3. Receiver
 - 휴대폰 하드웨어 레벨의 이벤트 리스너(ex 이어폰 연결)

4. Provider
 - 앱이 가진 데이터를 다른 앱에게 제공
